# Учебная библиотека top::pko412
![C++ Version](https://img.shields.io/badge/C++-11%20or%20above-blue?style=for-the-badge&logo=cplusplus)

Репозитарий содержит библиотеку учебных программ, используемых в рамках учебного процесса в компьютерной школе ТОП.
Библиотека использует пространство имен `top` для всех реализованных классов, функций и прочих элементов библиотеки.

## Состав модулей

### Класс `Point2d`

Файл: `point2d.h`

Класс, описывающий точку на двумерной координатной плоскости. Координатная плоскость целочисленная (координаты не могут принимать дробные значения).

### Класс `Rectangle`

Файл: `rectangle.h`

Класс, описывающий прямоугольник на двумерной координатной плоскости. Здесь применена агрегация: в составе полей класса - точки, представленные классом `Point2d`, реализованным в библиотеке.
